# מדריך עדכון משתמשים מקובץ Excel

## יצירת קובץ Excel

צור קובץ Excel בשם `users.xlsx` עם המבנה הבא:

### מבנה הקובץ:

| username | password |
|----------|----------|
| admin | admin123 |
| student1 | student123 |
| newuser | newpass123 |

### דרישות:

1. **שורה ראשונה** - כותרות העמודות: `username` ו-`password`
2. **רק שתי עמודות** - אין עמודות נוספות
3. **שם המשתמש** - חייב להיות ייחודי
4. **סיסמה** - תהיה מוצפנת אוטומטית לפני שמירה

## איך להשתמש

### אפשרות 1: העלאת קובץ

1. התחבר כמדריך (`admin` / `admin123`)
2. עבור ל-`/manage-users` או לחץ על "ניהול משתמשים מקובץ Excel" בלוח הבקרה
3. לחץ על "בחר קובץ" ובחר את קובץ ה-Excel שלך
4. הקובץ יעלה ויעודכן את המשתמשים אוטומטית

### אפשרות 2: סנכרון מקובץ users.xlsx

1. העתק את קובץ `users.xlsx` לתיקיית הפרויקט (בשורש)
2. התחבר כמדריך
3. עבור ל-`/manage-users`
4. לחץ על "סנכרן מקובץ users.xlsx"
5. המערכת תקרא את הקובץ ותעדכן את המשתמשים

## התנהגות המערכת

### משתמשים קיימים:
- אם שם המשתמש כבר קיים במערכת → הסיסמה תתעדכן
- שאר הנתונים (full_name, role) לא ישתנו

### משתמשים חדשים:
- אם שם המשתמש לא קיים → יווצר משתמש חדש
- `full_name` יוגדר אוטומטית לשם המשתמש
- `role` יוגדר אוטומטית ל-`student`

### הצפנת סיסמאות:
- כל הסיסמאות עוברות hash עם bcrypt לפני שמירה
- הסיסמאות לא נשמרות כטקסט פשוט

## דוגמה לקובץ Excel

צור קובץ Excel עם התוכן הבא:

```
username    | password
------------|------------
admin       | admin123
student1    | student123
student2    | student456
instructor1 | instructor123
```

## הערות חשובות

⚠️ **אזהרות:**
- ודא שהקובץ מכיל רק עמודות `username` ו-`password`
- שורה ראשונה חייבת להיות כותרות
- שם משתמש חייב להיות ייחודי
- סיסמה לא יכולה להיות ריקה

✅ **טיפים:**
- שמור גיבוי של המשתמשים לפני עדכון
- בדוק את התוצאות אחרי העדכון
- אם יש שגיאות, הן יוצגו בפירוט

## פתרון בעיות

### שגיאה: "הקובץ חייב להכיל עמודות: username ו-password"
- ודא שהשורה הראשונה מכילה בדיוק: `username` ו-`password`
- ודא שאין רווחים מיותרים בשמות העמודות

### שגיאה: "שם משתמש כבר קיים"
- זה לא שגיאה - המשתמש יתעדכן
- אם אתה רוצה ליצור משתמש חדש, שנה את שם המשתמש

### שגיאה: "קובץ users.xlsx לא נמצא"
- ודא שהקובץ נמצא בתיקיית הפרויקט (בשורש)
- או השתמש באפשרות העלאת קובץ
