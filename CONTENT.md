# מדריך תוכן - חוברת 44

## איך להוסיף תוכן למערכת

### הוספת תוכן מלא (מומלץ)

הרץ את הסקריפט:
```bash
npm run add-full-content
```

זה יוסיף את כל התוכן מהחוברת אוטומטית.

### מבנה התוכן

התוכן מחולק למודולים (נושאים):

1. **יסודות עזרה ראשונה** (3 שקפים)
   - מהי עזרה ראשונה
   - משולש החיים
   - המוח - מרכז השליטה

2. **הערכת מצב** (2 שקפים)
   - סיווג רמות הכרה
   - מדדים חיוניים

3. **החייאה** (4 שקפים)
   - החייאה למבוגר
   - החייאת ילד
   - החייאת תינוקות
   - דפיברילטור (AED)

4. **מצבי חירום נשימתיים** (2 שקפים)
   - חנק מגוף זר
   - אסתמה

5. **מצבי חירום רפואיים** (7 שקפים)
   - עילפון
   - אלרגיה ושוק אנפילקטי
   - הרעלות
   - אוטם בשריר הלב
   - סוכרת והיפוגליקמיה
   - שבץ מוחי
   - פרכוסים

6. **מצבי סביבה** (3 שקפים)
   - התייבשות
   - היפותרמיה
   - הכשת נחש

7. **טראומה** (4 שקפים)
   - מבוא לטראומה
   - פגיעות ראש
   - חבלה בעמוד שדרה
   - שברים בגפיים

## שאלות תרגול

לכל שקף יש לפחות שאלה אחת (חלקן עם כמה שאלות).

השאלות מבוססות ישירות על התוכן של השקף ועל החוברת.

## זמן קריאה

כל שקף כולל זמן קריאה מינימלי שנקבע אוטומטית לפי:
- אורך הטקסט
- מורכבות התוכן
- חישוב של ~200 מילים לדקה + בuffer של 50%

זמן מינימלי: 30 שניות

## הוספת תוכן נוסף

אם אתה רוצה להוסיף שקפים נוספים:

1. ערוך את `server/scripts/add-full-content.js`
2. הוסף את התוכן לפי הדוגמה הקיימת
3. הרץ: `npm run add-full-content`

### דוגמה להוספת שקף:

```javascript
slideId = await dbRun(
  `INSERT OR REPLACE INTO slides (module_id, title, content, min_reading_time, order_index) 
   VALUES (?, ?, ?, ?, ?)`,
  [
    module.id,  // ID של המודול
    'כותרת השקף',
    '<h2>כותרת</h2><p>תוכן השקף...</p>',
    calculateReadingTime('טקסט לשקף'),
    orderIndex  // מספר הסדר
  ]
);
```

### דוגמה להוספת שאלה:

```javascript
await dbRun(
  `INSERT OR IGNORE INTO practice_questions (slide_id, question, options, correct_answer, explanation) 
   VALUES (?, ?, ?, ?, ?)`,
  [
    slideId,
    'השאלה?',
    JSON.stringify({
      'א': 'תשובה א',
      'ב': 'תשובה ב',
      'ג': 'תשובה ג',
      'ד': 'תשובה ד'
    }),
    'א',  // התשובה הנכונה
    'הסבר למה זו התשובה הנכונה'
  ]
);
```

## תמונות וסרטונים

אם יש תמונות בחוברת:
1. שמור אותן ב-`public/images/slides/`
2. הוסף ב-`media_url` של השקף: `/images/slides/שם-התמונה.jpg`

דוגמה:
```javascript
await dbRun(
  `INSERT OR REPLACE INTO slides (module_id, title, content, media_url, min_reading_time, order_index) 
   VALUES (?, ?, ?, ?, ?, ?)`,
  [
    module.id,
    'כותרת',
    'תוכן',
    '/images/slides/mishulash-hayim.png',  // קישור לתמונה
    calculateReadingTime('טקסט'),
    1
  ]
);
```

## בדיקת התוכן

לאחר הרצת הסקריפט:

1. התחבר למערכת (admin/admin123)
2. עבור ל-/dashboard
3. לחץ על "התחל/המשך לימוד"
4. ודא שכל השקפים מופיעים
5. ודא שהשאלות מופיעות
6. ודא שזמן הקריאה נמדד נכון

## עדכון תוכן קיים

הסקריפט משתמש ב-`INSERT OR REPLACE` - זה אומר שאם תריץ אותו שוב, הוא יעדכן את התוכן הקיים.

אם אתה רוצה להסיר תוכן:
```sql
DELETE FROM practice_questions WHERE slide_id = X;
DELETE FROM slides WHERE id = X;
```

או בעזרת SQLite CLI:
```bash
sqlite3 database/learning_system.db
```
