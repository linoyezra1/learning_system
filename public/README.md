# תיקיית Public - קבצים להורדה

כדי לאפשר הורדת החוברת, יש להעתיק את הקובץ:

**"חוברת 44 מעודכן 04.01.pdf"**

לתיקייה זו (`public/`) או ל-`public/uploads/`

לאחר העתקה, החוברת תהיה זמינה להורדה דרך עמוד "חומרי לימוד להורדה".

### אפשרויות:

1. העתק את הקובץ ל-`public/חוברת 44 מעודכן 04.01.pdf`
2. או העתק ל-`public/uploads/חוברת 44 מעודכן 04.01.pdf`
3. או העתק ל-`public/handbook.pdf`

כל אחת מהאפשרויות הללו תעבוד.
