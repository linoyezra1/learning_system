# עדכון משתמשים מקובץ Excel

## יצירת קובץ users.xlsx

צור קובץ Excel בשם `users.xlsx` עם המבנה הבא:

### מבנה הקובץ:

| username | password |
|----------|----------|
| admin | admin123 |
| student1 | student123 |
| newuser | newpass123 |

### דרישות:

1. **שורה ראשונה** - חייבת להכיל: `username` ו-`password` (בדיוק כך)
2. **רק שתי עמודות** - אין עמודות נוספות
3. **שם המשתמש** - חייב להיות ייחודי
4. **סיסמה** - תהיה מוצפנת אוטומטית

## איך להשתמש

### שלב 1: יצירת הקובץ

1. פתח Excel (או Google Sheets)
2. צור טבלה עם שתי עמודות:
   - עמודה A: `username`
   - עמודה B: `password`
3. הוסף שורות עם משתמשים:
   ```
   username    | password
   admin       | admin123
   student1    | student123
   student2    | student456
   ```
4. שמור את הקובץ כ-`users.xlsx`

### שלב 2: עדכון המשתמשים

**אפשרות A: העלאת קובץ**
1. התחבר כמדריך (`admin` / `admin123`)
2. עבור ל-`/manage-users`
3. לחץ "בחר קובץ" ובחר את `users.xlsx`
4. הקובץ יעלה ויעדכן את המשתמשים

**אפשרות B: סנכרון מקובץ**
1. העתק את `users.xlsx` לתיקיית הפרויקט (בשורש)
2. התחבר כמדריך
3. עבור ל-`/manage-users`
4. לחץ "סנכרן מקובץ users.xlsx"

## התנהגות המערכת

- **משתמש קיים** → הסיסמה תתעדכן
- **משתמש חדש** → יווצר משתמש חדש עם role: student
- **הצפנה** → כל הסיסמאות עוברות hash עם bcrypt

## דוגמה לקובץ

ראה `users-template.csv` לדוגמה (ניתן להמיר ל-Excel)
